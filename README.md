## Requirements:

golang-1.20+

nvim 0.9.0+


## Setup

Clone the config

```
git clone https://github.com/dalpner/nvim-config ~/.config/nvim
```

Remove git folder
```
rm -rf ~/.config/nvim/.git
```

Start nvim with new configuration
```
nvim
```
